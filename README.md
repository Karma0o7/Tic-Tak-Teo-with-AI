# Tic-Tak-Teo-with-AI
A Tic-Tak-Teo game made in python to play against Computer that uses Alpha-Beta Pruning Algorithm

# Steps to run:-
1. Git clone https://github.com/Karma0o7/Tic-Tak-Teo-with-AI.git
2. python main.py
3. Play the game.

# Requirement:-
1. Python Compiler
2. pygame package (pip install pygames)
